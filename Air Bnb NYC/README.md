# Airbnb NYC Data Analysis

## Project Overview
This project explores the NYC Airbnb dataset (AB_NYC_2019). The goal is to understand pricing trends, availability, and host behaviors through data-driven exploratory analysis and visualization.

---

## Objectives
- Explore Airbnb listing data for New York City.
- Perform descriptive analytics on prices, locations, and room types.
- Identify host and neighborhood trends.
- Visualize spatial and categorical insights.

---

## Dataset
**File:** `AB_NYC_2019.csv`  
**Rows:** 48,896 (including header)

**Columns:**
- id, name, host_id, neighbourhood_group, neighbourhood, latitude, longitude, room_type, price, minimum_nights, number_of_reviews, last_review, reviews_per_month, calculated_host_listings_count, availability_365

---

## Files Included
- `Airbnb_NYC.ipynb` — main notebook containing data cleaning, analysis, and visualization.
- `AB_NYC_2019.csv` — dataset file.
- `The-NYC-Airbnb-Story.pptx` — summary presentation of key findings.

## Tools & Libraries
- Python 3.8+
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

---

## How to Run
1. Install required libraries:
   ```bash
   pip install pandas numpy matplotlib seaborn jupyterlab
   ```

2. Launch Jupyter Notebook:
   ```bash
   jupyter lab
   ```

3. Open `Airbnb_NYC.ipynb` and execute all cells.

---

## Key Insights
- Manhattan listings have the highest prices but limited availability.
- Entire home/apartment type dominates listings.
- Price variation correlates with neighborhood and room type.
